GrabDetector DLL exports functionality to detect grab\release gestures with OpenNI2
See Redistributable\Common\GrabDetector.h for full usage documentation


Redistributable:
----------------

The DLL, lib, include and Data folder
Copy the Data folder to the working directory of your final executible


Demo
----

Includes demo project to illustrate GrabDetector usage with ready-to-run executibles
Demo sources are also attached as VS2010 & VS2008 projects

The directory structure should be fully duplicated for the demos to compile
